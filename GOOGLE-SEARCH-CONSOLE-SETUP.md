# Google Search Console Setup for BertlClaw

## Cost
Google Search Console is free.
Sitemap.xml and robots.txt are free.

## Property to add
Recommended property URL prefix:

`https://bertlclaw.github.io/bertlclaw/`

## After adding the property
1. Open Google Search Console: https://search.google.com/search-console
2. Add property: `https://bertlclaw.github.io/bertlclaw/`
3. Use the recommended verification method Google offers for the URL-prefix property.
4. After verification, submit this sitemap:
   - `https://bertlclaw.github.io/bertlclaw/sitemap.xml`

## Important notes
- New GitHub Pages sites can take time to be indexed.
- Brand queries like `BertlClaw` often rank well once Google has crawled and indexed the site.
- If Google offers "Request indexing" for important pages, use it for:
  - `/`
  - `/services.html`
  - `/use-cases.html`
  - `/landingpages.html`
  - `/website-texte-positionierung.html`
  - `/ki-fuer-selbststaendige.html`
  - `/mvp-digitale-produktideen.html`
  - `/digitale-struktur-systeme.html`
